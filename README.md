## Lekcja 3

### Zadanie 3 - poziom podstawowy

Napisz funkcję `iterateAndSum`, która za pomocą metody `reduce` zsumuje wszystkie elementy przekazanej jako argument tablicy (array) i zwróci tą sumę (number).

Przykładowy input:
`iterateAndSum([1, 2, 3, 4]);`

Przykładowy output:
`10`
